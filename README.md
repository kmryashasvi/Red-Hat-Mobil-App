Purpose of this repository is to integrate Red-Hat and Docker in a Mobile App.<br>
* This is a Flutter project
* You can run all your commands of **Red Hat** and **Docker** in this app.

## Characteristics
* Made on Linux cmd
* Framework - Flutter
* Language - Dart
* Database - Firebase

## List of App Screens

* Main 
* Home Page
* Registration Page
* Login 
* Chat
